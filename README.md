# DayOfWeek
Реализуй метод getDayOfWeek(LocalDate date), чтобы он возвращал русское название дня недели аргумента date. Воспользуйся методами getDayOfWeek и getDisplayName.

Пример параметров метода getDisplayName, чтобы получить русское название дня недели:

getDisplayName(TextStyle.FULL, Locale.forLanguageTag("ru"))


псевдокод:

```
ПСЕВДОКОД: Просто день недели

ПЕРЕМЕННЫЕ:
    birthDate = Дата(2020, 3, 12)  // 12 марта 2020 года

ОСНОВНАЯ ПРОГРАММА:
    ФУНКЦИЯ main():
        деньНедели = getDayOfWeek(birthDate)
        ВЫВЕСТИ деньНедели
    КОНЕЦ ФУНКЦИИ

    ФУНКЦИЯ getDayOfWeek(дата):
        деньНедели = дата.получитьДеньНедели()
        текстовоеПредставление = деньНедели.получитьОтображение(
            стиль: ПОЛНЫЙ_ТЕКСТ,
            локаль: РУССКИЙ
        )
        ВЕРНУТЬ текстовоеПредставление
    КОНЕЦ ФУНКЦИИ
```

Пояснения:

1. Инициализация даты: Создается константная дата 12 марта 2020 года
2. Главная программа: Вызывает функцию получения дня недели и выводит результат
3. Функция getDayOfWeek:
   · Извлекает день недели из переданной даты
   · Форматирует его в полное текстовое представление на русском языке
   · Возвращает отформатированную строку

Ожидаемый результат выполнения: программа выведет "четверг" (поскольку 12 марта 2020 года действительно был четвергом)
